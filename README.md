Scripts to reproduce Fig. 2a are found in the `run` directory. 

Run `download.sh` to download and make LibLINEAR with data instance weights and to download and format the GIANT v1 brain-specific functional interaction network.

Run `run.sh` to train and evaluate models on several combinations of evidence categories. Results will be found in the `results` directory. The number of cross-validation folds and trials to run can be specified at the top of this script.
